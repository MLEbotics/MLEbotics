// MANUAL INSTALL — ZIP METHOD
4 Easy Steps (Chrome, Edge, Brave, Opera)
1

Download the Extension

Click the Download Extension button above. A file called autoformfiller-extension.zip will save to your Downloads folder.

2

Unzip the File

Find the ZIP in your Downloads folder. Right-click it → Extract All → pick a permanent folder (like your Desktop or Documents). Do not delete this folder after installing.

3

Open Chrome's Extensions Page

Open Google Chrome. In the address bar, type chrome://extensions and press Enter. In the top-right corner, flip the Developer mode toggle to ON.

Chrome address bar → chrome://extensions → Developer mode: ON ✓
4

Load the Extension & Add Your API Key

Click the Load unpacked button that appears, then select the folder you unzipped in Step 2. The 📋 Auto Form Filler icon will appear in your Chrome toolbar. Click it, go to the Settings tab, and paste your free Gemini API key.

✅ Free to use — Get a free Gemini API key at aistudio.google.com/app/apikey — takes 30 seconds, no credit card needed.
