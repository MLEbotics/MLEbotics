⚡ Auto Form Filler

Automatically detects the current page and fills in your personal data using AI.

🆕 Free tier

Uses Gemini Flash (Google) — no API key needed. Limited to 10 fills/day. For unlimited use, add your own free Gemini key at aistudio.google.com.

💰 Paid models

Claude (Anthropic) and GPT (OpenAI) require your own API key in My Data → API Keys. You pay your provider directly.

📄 What it fills

  • Name, email, phone
  • Address, city, state, ZIP, country
  • Company, job title, website
  • Date of birth, gender

🔒 Passwords & logins are handled by Bitwarden — install it separately.
