# ChromaShift

**System-level color accessibility overlay for Windows.**

ChromaShift remaps colors across your entire screen — any app, browser, or embedded site — in real-time, helping users with red-green color deficiency (deuteranopia / protanopia) see more clearly without any per-app configuration.

## Status

🚧 In Development

## Planned Features

- System-wide color filter overlay (no per-app setup required)
- Deuteranopia mode (green deficiency)
- Protanopia mode (red deficiency)
- Adjustable intensity / strength slider
- Hotkey toggle on/off
- System tray icon for quick access
- Works across all apps, browsers, games, and embedded sites

## Tech Stack

- Python / C++
- Windows System APIs (magnification layer / color matrix overlay)
